# explode

Multiplatform app developed using Flutter in course "Mobile Device Programming".

## How to use

Step 1:

Download or clone this repo by using the link below:
```bash
https://github.com/carvalho28/explode/
```

Step 2:

Get the required dependencies:
```bash
flutter pub get
```

Step 3:

Run the app:
```bash
flutter run
```
